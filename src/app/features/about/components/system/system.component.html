О системе

Веб-приложение “Классический кроссворд“ предоставляет большое количество возможностей для работы с кроссвордом.

Возможности администратора:

  • Создание, редактирование и опубликование кроссвордов
  • Подключение сторонних словарей понятий
  • Работа с базой данных системы

Возможности игрока:

  • Просмотр всех опубликованных кроссвордов
  • Добавление кроссворда в свою библиотеку
  • Решение любого кроссворда с системой подсказок
