Введение

Система создана для создания и решения классических кроссвордов. Это веб-приложение, рассчитанное на две основные группы пользователей: администраторов и игроков.

Администратор имеет доступ к созданию и редектированию кроссвордов, выборе режима заполнения кроссворда (автоматически или вручную), а также возможность добавить или удалить словарь понятий. Игрок может просматривать и выбирать кроссворд для решения из общей библиотеки, а также имеет возможность добавить в свою. Игрок также может разгадывать кроссворды с организацией системы подсказок.

Для корректной работы системы требуются определенные программные и аппаратные средства. На сервере необходима ЭВМ под управлением Windows Server 2016 или выше с установленной платформой Docker Desktop версии 3.3.0 и выше. Для клиентской части требуется ОС Windows 8, macOS 10.15 или Linux с графической средой, а также один из современных браузеров (Chrome 90+, Firefox 88+, Edge 91+, Safari 14+).

Работа с системой

Работа с системой в режиме администратора

На каждой странице администратора в верхней части экрана расположены кнопки "Словари", "Библиотека" и "Выйти", которые открывает страницу просмотра всех словарей, страницу со всеми кроссвордами из БД и производит выход из системы соответственно.

Вход в систему (авторизация)

Чтобы получить доступ к интерфейсу администратора необходимо перейти по адресу http://127.0.0.1/admin/, заполнить поля для авторизации и нажать кнопку «Вход». После успешной авторизации откроется главная страница администратора.

Просмотр всех кроссвордов

На главной странице показаны все кроссворды из БД, разбитые на страницы по 12 кроссвордов на каждой. Каждый кроссворд можно удалить или изменить. Также присутствует кнопка "Создать кроссворд" для создания нового кроссворда.

Создание нового кроссворда

Чтобы создать новый кроссворд нужно ввести начальные его параметры, такие как название, ширина и высота сетки, количество подсказок и выбрать используемый словарь понятий и способ заполнения кроссворда (автоматически или вручную). После этого нужно нажать на кнопку "Создать", при этом система проверяет корректность введенных параметров и если все корректно, то создает кроссворд.

Автоматическое заполнение кроссворда

Если способ заполнения кроссворда был выбран как автоматический, то система сама заполнит все слова в кроссворд на свое усмотрение. После этого система покажет всплывающее сообщение об успешном создании кроссворда и отобразить главную страницу администратора.

Ручное заполнение кроссворда

Если способ заполнения кроссворда был выбран как вручную, то система отобразить страницу ручного заполнения кроссворда, на которой администратору нужно выделить область на сетке, система построит для этого маску и выведет все слова, удолетворяющие маски в левой части страницы в выпадающем списке. Администратору нужно выбрать слово из этого выпадающего списка, после этого система запомнит это слово и отобразить его на сетке. Как только все слова введены, администратору нужно нажать на кнопку "Завершить", тогда система проверит корректность введенных слов. После этого система покажет всплывающее сообщение об успешном создании кроссворда и отобразить главную страницу администратора.

Изменение кроссворда

Если администратор выбрал изменить уже существующий кроссворд, то система отобразить страницу ручного заполнения кроссворда. Администратору также нужно будет проделать все шаги как в пункте "Ручное заполнение кроссворда".

Просмотр всех словарей понятий

На странице со словарями показаны все словари, загруженные в систему и разбитые на страницы.Администратор может удалить словарь, нажав соотвествующую кнопку рядом с выбранным словарем. Он также может добавить новый словарь понятий, нажав на кнопку "Добавить словарь".

Добавление нового словаря понятий

Если администратор нажал на кнопку "Добавить словарь", то система откроет страницу добавления нового словаря понятий, на которой предложит администратору выбрать файл на ПК. Далее администратору нужно нажать на кнопку "Подтвердить". Система проверит корректность файла, и в случае успеха отобразить страницу со всеми словарями понятий.

Работа с системой в режиме игрока

На каждой странице игрока в верхней части экрана расположены кнопки "Библиотека", "Мои кроссворды" и "Выйти", которые открывает страницу просмотра всех кроссвордов из БД, страницу с кроссвордами пользователя и производит выход из системы соответственно.

Вход в систему (авторизация)

Чтобы получить доступ к интерфейсу игрока необходимо пройти регистрацию или авторизацию, заполнить поля для и нажать кнопку «Вход». После успешной авторизации отроется главная страница игрока.

Просмотр всех кроссвордов

На главной странице показаны все кроссворды из БД, разбитые на страницы по 12 кроссвордов на каждой. Каждый кроссворд можно добавить в библиотеку игрока, для чего нужно нажать на соотвествующую кнопку рядом с выбранным кроссвордом.

Просмотр кроссвордов в библиотеке игрока

На странице с кроссвордами игрока показаны все кроссворды игрока, разбитые на страницы по 12 кроссвордов на каждой. Каждый кроссворд можно удалить из библиотеки игрока, для чего нужно нажать на соотвествующую кнопку рядом с выбранным кроссвордом. Также рядом с каждым кроссвордом написан процент решения этого кроссворда игроком.

Решение кроссворда

Если для решнеия был выбран кроссворд, то система отобразить страницу решения этого кроссворда, на которой игроку нужно выделить область на сетке. После этого система выделит определение к данному слову на левой части страницы, после чего игроку нужно ввести слово на сетку. Если слово было введено верно, то система зачеркнет определение этого слова на левой части страницы. Таким образом игроку нужно полностью решить кроссворд
